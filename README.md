IPL Cricket Analysis(2008-2024)

Firstly Collected, cleaned, and transformed IPL match and player performance data using Power Query to ensure accuracy and consistency. Then Analyzed historical match data to identify trends in team performance, player consistency, win/loss patterns, and key factors influencing match outcomes.After Analyzing Designed and developed interactive Power BI dashboards using DAX measures, calculated columns, and KPIs to evaluate batting,bowling, and overall team performance.Implemented slicers, filters, and drill-through features to enable dynamic analysis by season, team, venue, and player.
